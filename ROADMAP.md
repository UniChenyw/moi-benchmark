## 项目级里程碑

- **M0 / 本地结构基线**：五个根级 Track 和 `external/` 完成审阅。
- **M1 / 方案基线**：五个 Track 均有负责人和批准版本。
- **M2 / Track 试运行**：各 Track 形成首个可追溯 `run_id`。
- **M3 / 首个完整版本**：五份 Track 报告与综合索引通过 Release 冻结。
- **M4 / 持续回归**：任务与 evaluator 版本化，历史结果不被覆盖。
